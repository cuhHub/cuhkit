# Your Mod
This is your mod. Be sure to update `src/mod.xml`!

- To build your mod: `cuhkit build`
- To sync your mod to Stormworks (building does this too): `cuhkit sync`
- To publish your mod to cuhHub: `cuhkit publish`

Note that cuhkit will only find this project if executed from the same directory as the project `.json` file.
